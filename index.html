STEP 1

Dart 入門

プログラミングの基本をDartで学びます。難しい言葉は使いません。

Lesson 1 超基礎

プログラムってなんだろう？

コンピューターへの「命令書」のこと。料理レシピみたいに、順番に指示を書いていきます。

コンピューターは命令通りに動く

コンピューターはとても賢そうに見えますが、実は自分では何も考えられません。人間が書いた命令（プログラム）の通りにしか動きません。だからプログラムを書く = コンピューターへの指示書を作る、ということです。

Dartの最初の一歩

Dartプログラムは必ず main() という関数からはじまります。

Dart
void main() {
  print('こんにちは、世界！');
}
void main() → プログラムのスタート地点
print() → 文字を画面に表示する命令
{ } → 命令のまとまりを表す波カッコ
Lesson 2 超基礎

変数 — データを入れる箱

情報を一時的に保存しておく「名前付きの箱」のことです。

変数とは？

変数は「名前のついた箱」です。数字や文字など、いろんな情報を入れておけます。

変数の書き方

Dart
void main() {
  // 文字列（String）
  String name = 'たろう';

  // 整数（int）
  int age = 20;

  // 小数（double）
  double height = 170.5;

  // 真か偽か（bool）
  bool isStudent = true;

  print('名前: $name, 年齢: $age');
}
💡 ポイント
文字列の中に変数を入れるには $変数名 と書きます。これを「文字列補間」といいます。

varで型を省略できる

Dart
var name = 'はなこ'; // Dartが自動でStringと判断
var score = 100;    // Dartが自動でintと判断
Lesson 3 基礎

条件分岐 — もし〜なら

状況によって処理を変えたいときに使います。日常の「もし雨なら傘を持つ」と同じ考え方。

if文の書き方

Dart
void main() {
  int score = 75;

  if (score >= 80) {
    print('合格！よくできました');
  } else if (score >= 60) {
    print('もう少しがんばろう');
  } else {
    print('もっと勉強が必要です');
  }
  // → 'もう少しがんばろう' と表示される
}
if (条件) → 条件が「正しい」ときに実行
else if (条件) → 上が外れて、この条件が正しいとき
else → どれも当てはまらないとき
Lesson 4 基礎

繰り返し — ループ

同じ処理を何度も繰り返すときに使います。「10回挨拶する」を10行書かなくてよくなります。

forループ

Dart
void main() {
  // 1から5まで数える
  for (int i = 1; i <= 5; i++) {
    print('$i 回目！');
  }
}
// 出力:
// 1 回目！
// 2 回目！
// ...
// 5 回目！

リストと一緒に使う

Dart
void main() {
  List<String> fruits = ['りんご', 'みかん', 'ぶどう'];

  for (var fruit in fruits) {
    print(fruit);
  }
}
// 出力: りんご / みかん / ぶどう
Lesson 5 基礎

関数 — 処理をまとめる

よく使う処理に名前をつけて、いつでも呼び出せるようにしたものです。

関数の書き方

Dart
// 関数を定義する
String greet(String name) {
  return 'こんにちは、$name さん！';
}

void main() {
  // 関数を呼び出す
  String message = greet('たろう');
  print(message); // こんにちは、たろう さん！

  print(greet('はなこ')); // こんにちは、はなこ さん！
}
String（戻り値の型） → 関数が返す値の種類
greet → 関数の名前（自由につけられる）
String name → 引数（関数に渡す値）
return → この値を返す、という意味
Lesson 6 中級

クラスとオブジェクト

関連するデータや処理をひとまとめにする「設計図」のようなもの。これがわかるとプログラムがぐっと整理されます。

クラスとは？

たとえば「人間」というクラスを作るとき、人間には「名前」「年齢」があって「話す」「歩く」などの動作があります。クラスはこのような情報と動作をまとめる「設計図」です。

Dart
// クラスの定義（設計図）
class Person {
  String name;
  int age;

  // コンストラクタ（オブジェクトを作るときに呼ばれる）
  Person(this.name, this.age);

  // メソッド（クラスの中の関数）
  void introduce() {
    print('私は$nameです。$age歳です。');
  }
}

void main() {
  // オブジェクトを作る（設計図から実体を生み出す）
  var taro = Person('たろう', 20);
  var hanako = Person('はなこ', 18);

  taro.introduce();   // 私はたろうです。20歳です。
  hanako.introduce(); // 私ははなこです。18歳です。
}

Lesson 1 超基礎

Widgetって何？

FlutterはすべてWidgetでできています。テキスト、ボタン、画像…画面のあらゆる部品がWidgetです。

Flutterの世界観

Flutterのアプリは、Widgetを入れ子にして画面を作ります。まるでレゴブロックを積み重ねるように。

MyApp

Scaffold

Lesson 3 基礎

StatelessWidget vs StatefulWidget

Flutterには2種類のWidgetがあります。これを理解すると「動く」アプリが作れるようになります。

StatelessWidget

状態（変化）を持たないWidget。一度表示したら変わらない部品に使います。

例： タイトル文字、アイコン、固定のレイアウト

Dart
class MyTitle extends StatelessWidget {
  const MyTitle({super.key});

  Widget build(BuildContext context) {
    return Text('変わらないタイトル');
  }
}

StatefulWidget

状態（変化）を持つWidget。ボタンを押すと数字が増えるなど、変化する部品に使います。

例： カウンター、フォーム、チェックボックス

Dart
class Counter extends StatefulWidget {
  const Counter({super.key});
  State createState() => _CounterState();
}

class _CounterState extends State<Counter> {
  int count = 0;

  Widget build(BuildContext context) {
    return Column(children: [
      Text('$count'),
      ElevatedButton(
        onPressed: () => setState(() => count++),
        child: Text('+1'),
      ),
    ]);
  }
}
💡 setState() って何？
Flutterに「状態が変わったから画面を更新して！」と伝える命令です。setState() の中で変数を変えると、自動的に画面が再描画されます。
Lesson 4 基礎

レイアウトの基本

Column・Row・Stack・Paddingを組み合わせて、思い通りの画面を作りましょう。

Padding — 余白をつける

Dart
Padding(
  padding: EdgeInsets.all(16),   // 四方16pxの余白
  child: Text('余白つきテキスト'),
)

Column + mainAxisAlignment — 縦並びの整列

Dart
Column(
  mainAxisAlignment: MainAxisAlignment.center,
  crossAxisAlignment: CrossAxisAlignment.start,
  children: [
    Text('1行目'),
    SizedBox(height: 8),   // スペーサー
    Text('2行目'),
  ],
)

Expanded — 残りスペースを埋める

Dart
Row(
  children: [
    Expanded(child: Text('左側（残り全部）')),
    Text('右側（必要な分だけ）'),
  ],
)

実践：はじめてのアプリ

学んだことを使って、実際に動くカウンターアプリを作りましょう。

📱

カウンターアプリ

ボタンを押すたびに数字が増えるシンプルなアプリ。Flutterの基本がすべて詰まっています。

Step 1

Flutter SDKをインストール

flutter.dev から自分のOS用のSDKをダウンロードしてインストールします。

Step 2

VS Codeをインストール

コードを書くエディタ。code.visualstudio.com からダウンロード。Flutter拡張機能も入れましょう。

Step 3

新しいプロジェクトを作る

ターミナル
flutter create my_counter_app
cd my_counter_app
flutter run
Step 4

コードを書こう

lib/main.dart を以下のコードに書き換えてみましょう。

lib/main.dart
import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'カウンターアプリ',
      theme: ThemeData(colorSchemeSeed: Colors.blue),
      home: const CounterPage(),
    );
  }
}

class CounterPage extends StatefulWidget {
  const CounterPage({super.key});
  State createState() => _CounterPageState();
}

class _CounterPageState extends State<CounterPage> {
  int _count = 0;

  void _increment() {
    setState(() => _count++);
  }

  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('カウンター')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            const Text('ボタンを押した回数'),
            Text(
              '$_count',
              style: Theme.of(context).textTheme.displayLarge,
            ),
          ],
        ),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: _increment,
        child: const Icon(Icons.add),
      ),
    );
  }
}
