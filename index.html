プログラムってなんだろう？

変数 — データを入れる箱

情報を一時的に保存しておく「名前付きの箱」のことです。

変数の書き方

Dart
void main() {
  String name = 'たろう';   // 文字列
  int age = 20;             // 整数
  double height = 170.5;   // 小数

💡 ポイント
文字列の中に変数を入れるには $変数名 と書きます。これを「文字列補間」といいます。

繰り返し — ループ

同じ処理を何度も繰り返すときに使います。

forループ

Dart
void main() {
  for (int i = 1; i <= 5; i++) {
    print('$i 回目！');
  }
}
int i = 1 → iを1からスタート
i <= 5 → iが5以下の間くり返す
i++ → 1回終わるたびにiを1増やす

クラスとオブジェクト

関連するデータや処理をひとまとめにする「設計図」のようなもの。

クラスの書き方

Dart
class Person {
  String name;
  int age;

  Person(this.name, this.age);

  void introduce() {
    print('私は$nameです。$age歳です。');
  }
}

void main() {
  var taro = Person('たろう', 20);
  taro.introduce();  // 私はたろうです。20歳です。
}

Flutter 入門

Dartの基礎ができたら、いよいよFlutterでアプリを作ります。FlutterのUIはDartPadで動かせます。

Lesson 1 超基礎

Widgetって何？

FlutterはすべてWidgetでできています。テキスト、ボタン、画像…画面のあらゆる部品がWidgetです。

最初のFlutterアプリ

Dart (Flutter)
import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('最初のアプリ')),
        body: const Center(
          child: Text('こんにちは、Flutter！'),
        ),
      ),
    );
  }
}

Lesson 3 基礎

StatelessWidget vs StatefulWidget

Flutterには2種類のWidgetがあります。これがわかると「動く」アプリが作れます。

StatelessWidget

変化しない部品。タイトル文字やアイコンなど。

StatefulWidget

変化する部品。カウンター、フォームなど。setState() で画面を更新。

カウンターアプリ（StatefulWidget）
class _CounterState extends State<Counter> {
  int count = 0;

  Widget build(BuildContext context) {
    return Column(children: [
      Text('$count'),
      ElevatedButton(
        onPressed: () => setState(() => count++),
        child: Text('+1'),
      ),
    ]);
  }
}
